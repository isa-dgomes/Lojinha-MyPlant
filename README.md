#LOJINHA- MyPlant
Esse repositório tem o simples intuito de compartilhar meus avanços no estudo de Java, em listas e classes.
O projeto tem como tema uma lojinha de plantas, pra calcular preço das compras, calcular troco e interagir com as "pessoas".
Ele é composto pelas classes Vendedor, Cliente, Loja e vendas.
Meus principais avanços foram no uso de localDate e manipulação das listas entre as classe.
